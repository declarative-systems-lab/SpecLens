# SpecLens Website

Project website for **SpecLens**.

Built with Astro and a small React island for the interactive configuration explainer.

## Website

Visit the SpecLens website directly:

https://declarative-systems-lab.github.io/SpecLens/

## Development

```bash
cd website2
npm install
npm run dev
```

Open http://localhost:4321

## Build

```bash
npm run build
```

Output is in `dist/`.

The user-study app is at https://declarative-systems-lab.github.io/UserstudyWeb/ — link only.
